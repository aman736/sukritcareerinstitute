# Sukrit Career Institute – Website

This repository contains the source code for the official website of **Sukrit Career Institute**, hosted using **GitHub Pages**.

The website provides basic information about the institute, courses offered, and contact details.

---

## 📘 About Sukrit Career Institute

Sukrit Career Institute is an academic coaching institute focused on:

- Foundation courses for **Class 9 and Class 10**
- Board examination preparation for **Class 11 and Class 12**
- Competitive exam preparation for:
  - **IIT‑JEE**
  - **Medical (NEET)**

The institute emphasizes concept clarity, structured learning, and exam‑oriented preparation.

---

## 🎯 Courses Covered

- Foundation (Class 9 & 10)
- Senior Secondary (Class 11 & 12 – Science)
- IIT‑JEE (Engineering Entrance)
- Medical Entrance Preparation (NEET)

---

## 🌐 Website Hosting

- Hosting Provider: **GitHub Pages**
- Type: **Static website**
- HTTPS: **Enabled by default**
- Custom Domain: Supported

The site is built using simple HTML/CSS and does not use any server‑side technology.
